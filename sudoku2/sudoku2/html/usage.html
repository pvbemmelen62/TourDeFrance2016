General info

This program allows you to enter a sudoku of size 9 or size 16, and then you can click a button to start the search for solutions to the sudoku.

This program handles only sudokus of the simple variant:

  • each column must contain the numbers 1 ... size;
  • each row must contain the numbers 1 ... size;
  • each subrectangle must contain the numbers 1 ... size; for sudoku size 9, the subrectangles have size 3; and for sudoku size 16, the subrectangles have size 4.

While the program is searching for a solution, it shows the numbers that it is trying.

How to enter a sudoku numbers

To enter a new sudoku, select File->New... ; next choose the size; and then enter numbers from the keyboard. Those numbers will be fixed during the search for a solution.

You can only enter numbers while the search for a solution has not yet started; i.e. "step count" must be 0. When you can enter numbers, the drawing will show a cursor (a small vertical line) in the cell where the number will go.

You can use the arrow keys to move the cursor, the digit keys to enter numbers, and the backspace key to delete a digit.

If you have clicked one of the toggles (see below), the drawing where you enter numbers will have lost the keyboard focus; you can tell because the cursor, will be gone. In that case click on the drawing, to give it keyboard focus again.

While entering numbers, the program will block any input that violates the sudoku rules. This means that if you entered "1" somewhere in a row, that you can not enter "12" any more, because to enter "12", you first need to type "1" and then "2".
To get around this problem, enter "1" only after you have entered any "10", "11", ... "16".

Search for solutions

You can click on the "Step" button, and the program will perform one step forward in its search for a solution.

Or you can click the "Start" button, and the program will keep on stepping until it has found a (next) solution, or until it has found that there is no more next solution.
The "Start" button, when clicked, will change into a "Stop" button so that clicking it again, will stop the search.

The speed with which the program searches a solution, can be reduced by increasing the delay with the horizontal slider control.

For fastest speed, choose delay 0. Then, the speed with which the program searches a solution, will be higher if the program does not update the drawing of the sudoku at every step.

Therefore, the program allows you to disable the redrawing, by disabling the "Refresh cells" toggle.
When you click the "Step" button for a single step, the redrawing always occurs, irrespective of the toggle.

Similarly, you can disable the redrawing of the "step count".

When you have disabled both toggles, and you click the "Start" button, you can not directly see the progress.
But you can see when the program has found a solution, or has found that there is no solution, by looking at the "Start/Stop" button. While the program is searching the button will have label "Stop" ; and when the search has stopped, the label will show "Start" again.
